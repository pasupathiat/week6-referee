## Problem
Institutions struggle to choose the right student entry detection system due to trade-offs between cost, accuracy, and maintenance.

## Solution
This web-based tool compares:
- IR Sensor
- RFID System
- Camera-based System

Based on user priority, the tool explains pros, cons, and recommends the best option.
